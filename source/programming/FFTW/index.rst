FFTW Learning Notes
===============================================

本文档是学习FFTW 3.3 的学习笔记，主要基于 `fftw3.3官方的手册 <http://notes.yeshiwei.com/fftwdoc/html/>`_ 。
..
   在线版本可以在下面链接看到::

    http://notes.yeshiwei.com/fftw/

   sphinx 源码可以从svn上获取::

    svn co svn://lab.yeshiwei.com/svnrepos/fftw/

   从源码中获得tex，在根目录::

    make latex
    python xelatex.py

   然后可以直接到latex目录下make 获得PDF::

    cd _build/latex/
    make

   下载 PDF::

     http://notes.yeshiwei.com/fftw/FFTWLearningNotes.pdf
